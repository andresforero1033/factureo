# Factureo - Sistema Financiero Integral

Módulo de Recursos Humanos y Nómina construido con arquitectura modular.

## Stack Tecnológico
- **Backend:** Python, FastAPI, Motor (MongoDB Async)
- **Base de Datos:** MongoDB
- **Frontend:** Vanilla JS (ES6 Modules), HTML5, CSS3

## Ejecución Local
1. **Backend:** \`cd backend\` -> \`pip install -r requirements.txt\` -> \`uvicorn app.main:app --reload\`
2. **Frontend:** Servir la carpeta \`frontend\` con un servidor estático (ej. Live Server o \`python -m http.server 5500\`).
